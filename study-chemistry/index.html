必修一

  • 双水解相关问题：

  1. Al3+与HCO3-、CO32-、HS-、S2-

  2. Fe3+与HCO3-、CO32-

  3. SiO32-、AlO2-与所有能水解的阳离子

  4. NH4+与SiO32-、AlO2-

    （摘自 Wikipedia）

  • FeS 与一定浓度的 HNO3 反应，生成Fe(NO3)3、Fe2(SO4)3、NO2、N2O4、NO 和H2O，当NO2、N2O4、NO 的物质的量之比为1：1：1 时，实际参加反应的 FeS 与 HNO3 的物质的量之比为？
    解：不妨设n(NO2) = n(N2O4) = n(NO) = 1 mol
    由得失电子守恒：1+2+3 = n(FeS)(1+8) ⇒ n(FeS) = 2/3 mol
    **    由 S 守恒**：n[Fe2(SO4)3] = n(FeS) / 3 = 2/9 mol
    进而：n[Fe(NO3)3] = n(FeS) -2n[Fe2(SO4)3] = 2/9 mol
    故：n(HNO3) = n(NO2) + 2n(N2O4) + n(NO) + 3n[Fe(NO3)3] = 1+2+1+2/3 = 4+2/3 mol
    n(FeS) : n(HNO3) = 2/3:(4+2/3) = 1 : 7
